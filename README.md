# AI Recruitment System

## Overview
This project is an AI-powered Recruitment System that helps analyze candidate profiles and recommend suitable job roles based on their skills, education, and experience.

## Features
- Resume Analysis
- Job Recommendation
- Candidate Ranking
- Skill Gap Analysis
- Interview Question Generation

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit

## Dataset
The project uses a JSONL dataset containing candidate profile information.

## Installation
```bash
pip install -r requirements.txt
```

## Run the Project
```bash
streamlit run app.py
```
